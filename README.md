# toplu-tasima-guzergah-planlayici
 Bu proje, toplu taşıma güzergahlarını planlamak için Softmax algoritmasını kullanır. Farklı mahalleler, nüfus yoğunluğu, mevcut ulaşım altyapısı, maliyet analizi, çevresel etki ve sosyal fayda gibi kriterlere göre değerlendirilir. Algoritma, her mahalle için uygunluk skorlarını hesaplar ve en uygun toplu taşıma güzergahlarını önerir.
